# ae_star_wars_searcher

Star Wars Searcher API

## Requirements

[poetry >= 1.1.13](https://python-poetry.org/docs/#installation)

## Installation 

```sh
poetry install
```

## Usage

### Run Test Coverage

```sh
make coverage
```

### Run Formatter

```sh
make format
```

### Run Linter

```sh
make lint
```

### Run Tests

```sh
make test
```
